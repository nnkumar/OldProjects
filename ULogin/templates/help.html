UVLogin


Universal Login enables you to access all our accounts with a single signin.
You dont need to remember large number of username and passwords.
Will save you the time to open the login page and type in details.
Multiple accounts with the same website can be easily managed.


Getting Started


Using UVLogin is trivail. If u are a registered user login to our site at www.uvlogin.com/login else register at www.uvlogin.com/signup.
Select a user name and password and create an account.

Adding a Site


After typing your username and password you are redirected to the home page.




Here you can sides in which you want login automatically without providing usernames and passwords each time. Click on the add site button seen on the above image(available both in the top panel of all pages and the left bottom of home page). Then will be directed the page shown below


Here you need to fill four things

1.URL

Here you should add the address of the login page of the site which you wish to login automatically. eg: www.uvlogin.com/login/

2.Display Name

This is an optional field. Here you can select a name for the site to which enabling automatic login facilty. eg: uvlogin if you want login to the site www.uvlogin.com.
If this field is left black the diplay name will be chosen as the same of  URL

3.Username

Here you should provide the username of the site which u want to login automatically.

4.Password

Here you should provide the password of the site to which you want to login automatically.

After filling this fields click the submit button.

Logging in to sites Automatically







Once you have added sites to your account, the auto-login(automatic login) feature  will be available on your home page. As all the sites which you have added will be displayed. To login to the respective site you can click the signin button. On clicking the signin  a new window will be opened  and you will be logged in auutomatically to the correspondin site.

To change the username,password or display name you can use the edit button. To delete a  added site u can use the delete button.

SL2Login

Security level 2 login of this site provides you with enhanced security to protect from Keyloggers . Keyloggers are programs which runs in the background and saves what ever we type in (including our passwords) and which will be accessable to the one who installed it.This way we are in danger of losing our accounts when we access internet in insecure systems like Internet Cafes. (These accounts can even be misused for terrorist activities)

        Protects from keyloggers
       
        Useful while travelling and in internet cafes.

Enabling SL2Login

Inorder to use SL2Login facility you need to

1.Login to UVLogin if you are already an user else create an account.

2.Select the settings tab from the home page.



3. Enter your UVLogin password.

4. Select a pin numeric for your SLLogin.(It cannot be an alphabet)

5. Click Submit button. SL2Login facility is available for your account now.


Logging in using SL2Login


1.Click the SL2Login button the home page.

2.In the login name filed type your UVLogin username.

3.Now say the pin you set is 1234.(Look the Enabling part to find how to set the Pin). Inorder to findout the key look the mapping on the code box. 1 is mapped to t ,2 to h, 3 to m and 4 to i.

so your key is "thmi". This key changes from time to time  you are loging in .So always check the mappings in the code box to get the key for your current session.

4 After entering the key type the Login button.You will be redirected to your home page.
